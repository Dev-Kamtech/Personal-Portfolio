# Personal-Portfolio

A vibrant and interactive personal portfolio website showcasing my skills, projects, and experience as a developer.

## Features

- **Interactive Hero Section** - Animated introduction with bouncing avatar and floating decorations
- **Skills Showcase** - 8 clickable skill cards with dedicated learning pages for Flutter, Dart, HTML, CSS, JavaScript, Firebase, Supabase, and React Native
- **Project Portfolio** - 6 featured projects including Weather App, Currency Converter, Expense Tracker, Quote Generator, Tic-Tac-Toe, and WorkPost App
- **Responsive Design** - Fully responsive layout that works on desktop, tablet, and mobile devices
- **Smooth Animations** - Scroll reveal animations, hover effects, and interactive elements
- **Draggable Elements** - Floating emojis that can be dragged around the screen
- **Auto-generating Sparkles** - Dynamic sparkle animations throughout the page

## Technologies Used

- HTML5
- CSS3 (with modern animations and gradients)
- Vanilla JavaScript
- Responsive Design

## Getting Started

Simply open `index.html` in your web browser to view the portfolio locally.
